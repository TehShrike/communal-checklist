Why?

My wife wanted an informal gift registry that wasn't tied to a particular outlet. All the options that she found were lame in some way.

How

  1. Click the big button above to create a new list
  2. Type in a name for your list
  3. Add your list items
  4. Add more checkboxes to the items (in case you want six pairs of socks or something, so that different people can check as many of the checkboxes as they bought)
  5. If you want, you can edit the items to add a link to an online listing of the item you have in mind
  6. Copy the "share" link to send to your friends
  7. Bookmark the current page so that you can get back to the list and be able to edit it later if you need to

Anything else?

  • Anyone with the long URL can edit everything about the list
  • The checkboxes aren't permission controlled - they are associated with the name that people typed in, but that's only there to help people remember what they checked before - if you type in someone else's name, you'll be able to uncheck whatever they checked
  • If you have any questions, want to run the server yourself, or want to discuss a feature, check out the project on Github - the whole thing's open source
